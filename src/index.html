使用说明

  • 选择一个杂志模板作为风格参考
  • 上传你的图片（人物或物品照片效果最佳）
  • 点击"生成小红书封面"按钮
  • 系统会根据模板风格重新设计你的图片
  • 生成的封面将保留模板的排版、字体和设计风格

设置 Gemini API Key

如何获取 API Key：
1. 访问 Google AI Studio (https://makersuite.google.com/ )
2. 创建或选择一个项目
3. 点击 "Get API Key" 获取密钥
